# Hiddify FAQ на русском

Решения повторяющихся проблем по Hiddify VPN - установка, настройка панели, маршрутизация, WARP, клиенты.

## Что внутри

Слева в навигации - разделы по темам. Сейчас доступно:

- **Установка и обновление** - порядок шагов на новом сервере, выбор VPS и домена, IPv6, порты и файрвол, грабли свежей установки и автообновления.
- **Клиент и подключение** - установка клиента, iPhone без App Store, DNS, раздельное туннелирование, "подключено но нет интернета", режим VPN/Proxy в Windows.
- **Hiddify Panel** - установка и 503, зависания установщика, "0 пользователей онлайн", proxy-loop из-за Decoy Site.
- **VLESS+Reality** - настройка, выбор протокола под российский DPI.
- **WARP** - подключение и устранение проблем с WARP/WARP+ для Google, Gemini, YouTube.
- **Cloudflare** - edge-IP под РКН, CDN-домены, Universal SSL, разница Wi-Fi и мобильного интернета.
- **Sing-box / Xray** - выбор ядра, шаблоны .pj2, баг nginx error_log off, apply_configs.
- **Mihomo-каскад** - цепочка Hiddify через mihomo.
- **MTProto** - дрейф faketls-домена, telemt против mtg.

Список будет расти.
