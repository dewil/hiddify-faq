# Hiddify FAQ на русском

Решения повторяющихся проблем по Hiddify VPN - установка, настройка панели, маршрутизация, WARP, клиенты.

## Что внутри

Слева в навигации - разделы по темам. Сейчас доступно:

- **WARP** - подключение и устранение проблем с WARP/WARP+ в Hiddify.
- **VLESS+Reality** - настройка, типичные грабли, SNI, "подключено но нет интернета".
- **Hiddify Panel** - установка, 503, зависания установщика, silent fail Apply Config.
- **Cloudflare** - edge-IP под РКН, CDN-домены, Worker-ошибки.
- **Sing-box / Xray** - выбор ядра, CPU 100% на xray, несовместимые транспорты.

Список будет расти.

## Сообщество

Материал собран из обсуждений в [@hiddify_board](https://t.me/hiddify_board) - международном чате разработчиков Hiddify, и пересказан своими словами.

## Как пользоваться

В верхней панели справа - поиск по сайту. Каждая страница - один симптом или один сюжет, читается за минуту.
