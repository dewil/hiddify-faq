Новая версия панели иногда ломает установку, применение настроек или работу с доменами - тогда хочется вернуться на ту, что работала. Механика выбора версии в Hiddify Manager стабильна и не зависит от конкретного релиза.

## В чем суть

Панель Hiddify Manager ставится и обновляется одним скриптом из официального репозитория `github.com/hiddify/Hiddify-Manager`. Какая версия установится, определяет аргумент канала, который вы передаете установщику. По умолчанию ставится последний релиз из канала "release" (стабильный). Беты живут в отдельном канале "develop". Можно явно указать и конкретный тег версии - тогда установится именно она, а не "последняя".

## Как откатиться на нужную версию

Откат - это просто повторная установка скрипта с указанием нужной версии. Данные пользователей и настройки при переустановке сохраняются, но бэкап перед откатом сделать обязательно.

Стабильный (последний релиз):

```bash
bash <(curl https://i.hiddify.com/release)
```

Конкретная версия передается установщику тегом вида `v11.0.0`. Точный синтаксис аргумента сверьте в актуальной документации или в меню панели - в репозитории теги доступны в разделе Releases, а установщик умеет ставить по тегу `vX.Y.Z`. Не угадывайте номер: возьмите тег той версии, на которой у вас все работало.

## Каналы: стабильный vs develop

Стабильный канал (release) - проверенные релизы, дефолт для боевых серверов. Канал develop (бета) - свежие, но сырые сборки; туда заглядывают, только если в стабильном баг, который уже починили в бете:

```bash
bash <(curl https://i.hiddify.com/develop)
```

Переключение между каналами - это просто запуск соответствующего скрипта. На рабочем сервере держите стабильный канал.

## Отключить автообновление

Чтобы версия не менялась сама, отключите автообновление в настройках панели (раздел с обновлениями в админке) - там есть переключатель. На уровне системы автообновление обычно запускает задача в cron; если переключателя в панели не нашли, проверьте и при необходимости отключите соответствующую cron-задачу на сервере. Конкретное расположение опции уточните в меню вашей версии - формулировки между релизами слегка отличаются.
