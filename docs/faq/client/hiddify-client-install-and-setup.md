# Установка и настройка клиента Hiddify

Hiddify - клиент для доступа к зарубежным сервисам через иностранный IP. Российские сайты при этом идут напрямую, без туннеля, переключать соединение вручную не нужно. Работает в режиме VPN-сервиса (рекомендуется) или системного прокси.

## Что нужно

- Устройство с Windows, macOS, Android или iOS.
- Ссылка-подписка на ваш профиль (ее выдает администратор сервиса). Выглядит так: `https://vpn.example.com/<path>/sub`.

## Шаги

### Шаг 1. Установить клиент под свою ОС

- iOS, iPadOS, macOS - Hiddify в App Store.
- Windows, Linux - релизы Hiddify на GitHub.
- Android (в том числе Android TV) - Google Play или APK с GitHub.

### Шаг 2. Импортировать профиль

Запустите приложение (на десктопе лучше от имени администратора). Нажмите "Новый профиль" (плюсик) и вставьте ссылку-подписку из буфера обмена.

### Шаг 3. Включить режим VPN-сервис

В режиме работы выберите "VPN Сервис" - так туннель охватывает весь трафик системы, а не только браузер.

### Шаг 4. Настроить раздельное туннелирование

В "Параметры конфигурации":

- "Регион" - "Россия (ru)". Локальный российский трафик пойдет напрямую, мимо туннеля.
- "Маршрутизация IPv6" - "Отключено". Иначе часть трафика может утечь мимо VPN.

### Шаг 5. Задать Remote DNS

В разделе "Параметры DNS" поле "Удаленный DNS" переключите на DoH-резолвер, например `https://1.1.1.1/dns-query`.

## Проверка

Сравните свой IP до и после подключения - откройте два сайта:

- Российский сервис проверки IP - должен показать IP вашего провайдера.
- Зарубежный сервис проверки IP - должен показать IP за пределами РФ.

Если IP разные - все настроено правильно. Если оба показывают ваш домашний IP, туннель не поднялся: проверьте, что включен VPN-сервис, и обновите подписку (кнопка с круговыми стрелками рядом с профилем).
