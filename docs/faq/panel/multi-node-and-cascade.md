# Несколько серверов под одной панелью и каскад - что умеет Hiddify

Два частых запроса, которые путают между собой. Оба про "несколько серверов", но это разные задачи, и штатно Hiddify Manager не решает ни ту, ни другую - панель рассчитана на один сервер.

## Мультинода: несколько локаций под одной панелью

Запрос: объединить два-три VPS в разных странах под одной панелью, чтобы пользователи и подписки были общими, а клиент мог переключать локацию. В Marzban для этого есть "ноды", в Hiddify Manager штатной мультиноды нет - каждая установка управляет только своим сервером.

Что делают на практике:

- **Агрегированная подписка.** Несколько отдельных панелей Hiddify (по одной на сервер), а клиенту отдается одна ссылка-подписка, которая собирает конфиги со всех. Клиент видит список локаций и переключается между ними. Сведение подписок - на стороне внешнего агрегатора, не самой панели.
- **Синхронизация пользователей по API.** У панели есть API - пользователей на нескольких серверах держат в синхроне внешним скриптом, а не вручную в каждой панели.
- **Другая панель.** Если мультилокация - ключевое требование, часто берут панель, где ноды есть из коробки (Marzban, Remnawave, 3x-ui), а Hiddify оставляют там, где хватает одного сервера.

## Каскад: вход в одной стране, выход в другой

Запрос: клиент подключается к серверу в стране A, а трафик выходит в интернет через сервер в стране B (вход на "чистом" или белосписочном IP, выход за рубежом). Сам Hiddify Manager цепочку между серверами не строит - это делается на сервере, ниже уровня панели:

- Перенаправлением исходящего трафика ноды через локальный прокси-цепочку (mihomo/sing-box). Серверная настройка такой цепочки описана отдельно: [Hiddify через mihomo - настройка цепочки](../mihomo/hiddify-via-mihomo.md).
- TCP-проксированием на уровне HAProxy между входной и выходной нодой.

В обоих случаях панель об этом не знает - она видит обычный сервер, а межсерверный прыжок настроен в обход нее.

## Коротко

- Несколько локаций в одной подписке - не панель, а внешний агрегатор подписок или альтернативная панель с нодами.
- Каскад вход-выход - серверная цепочка (mihomo/HAProxy), не функция панели.
- Если нужна именно штатная мультинода - Hiddify Manager сегодня не тот инструмент.
